Personal Projects

JSwiki

JSwiki is (going to be) a wiki based on javascript and able to run without any kind of webserver, but just on your local machine.

I wanted to be able to maintain a wiki on my laptop, without having to be connected to the internet. I didn't want to run a webserver just for that, since it would interfere with the development work I have to do. I did not want a desktop app, but something that feels like I work online. And I still wanted to be able to edit pages with my text editor.

If you feel the same way, have a look at my work and feel free to contribute: GitHub Repository

TiddlyIO

For JSwiki, I needed to access files on the local machine with JavaScript. TiddlyWiki does this, so I started building a library around their functionality.

Feel free to use this library for your projects and contribute back to it: GitHub Repository

github-img-badge

The GitHub-Image-Badge is a php script which creates an image similar to the output of the github-commit-badge. Usefull to integrate in forums etc., where JavaScript or even plain HTML is not allowed, but images are.
