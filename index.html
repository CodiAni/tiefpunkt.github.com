Personal Projects

VPNmon

I run a small VPN service based on OpenVPN for a number of friends. After a while, I was curious as to how much data everyone transfers across that VPN. Looking for a webbased tool to monitor those VPN tunnels, I couldn't find any. That was when I started to build VPNmon.

VPNmon is written in PHP and built on top of the Symfony Framework. It enables you to monitor multiple VPN servers running on the same machine, and can give you information about currently logged on users, and also past VPN sessions and summaries thereof.

A number of extensions is planned, and will be added as time permits. If you want to contribute, or have some wishes as to which functions you want to have implemented, hit up the GitHub Repository and let me know.

MokuWiki

MokuWiki is a wiki based on javascript and able to run without any kind of webserver, but just on your local machine.

I wanted to be able to maintain a wiki on my laptop, without having to be connected to the internet. I didn't want to run a webserver just for that, since it would interfere with the development work I have to do. I did not want a desktop app, but something that feels like I'm working online. And I still wanted to be able to edit pages with my text editor.

github-img-badge

The GitHub-Image-Badge is a php script which creates an image similar to the output of the github-commit-badge. Usefull to integrate in forums etc., where JavaScript or even plain HTML is not allowed, but images are.
